淺拷貝（Shallow Copy）： 淺拷貝是指僅複製物件的第一層屬性，而不會複製物件中的嵌套物件的內容。 當你進行淺拷貝時，原始物件及其屬性被複製到新的物件中，但如果屬性是一個物件或陣列，則新物件中的該屬性將只是對原始物件中該屬性的參考。 在JavaScript中，常見的淺拷貝方法包括使用Object.assign()或展開運算子（...）。 // 使用 Object.assign() 進行淺拷貝 深拷貝（Deep Copy）： 深拷貝是指將整個物件以及所有嵌套的物件都複製到新的物件中，而不僅僅是第一層屬性。 深拷貝會遞歸地複製物件中的所有內容，確保新的物件與原始物件完全獨立，對其中一個物件的修改不會影響到另一個。 由於JavaScript中的物件可以是相互嵌套的，因此實現深拷貝可能會比較複雜。常見的方法包括遞歸複製每個屬性或使用第三方庫（如lodash）提供的深拷貝函數。
